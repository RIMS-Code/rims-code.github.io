# Schemes for Iron

## Element information

- **Symbol:** Fe
- **Series:** TransitionMetal

- **Atomic weight:** 55.845 u
- **Ionization potential:**  63737.704 cm⁻¹ ([NIST ASD, 2024](https://www.nist.gov/pml/atomic-spectra-database))

- **Boiling point:** 3134.2 K
