# Schemes for Strontium

## Element information

- **Symbol:** Sr
- **Series:** AlkalineEarthMetal
- **Atomic number:** 38
- **Atomic weight:** 87.620 u
- **Ionization potential:** 45932.204 1/cm
- **Melting point:** 1050.2 K
- **Boiling point:** 1655.2 K
